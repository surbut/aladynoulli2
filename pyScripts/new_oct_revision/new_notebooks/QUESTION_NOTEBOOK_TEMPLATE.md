# Template for Reviewer Question Notebooks

## Structure

Each question notebook should follow this structure:

1. **Header** - Question, reviewer, status
2. **Context** - What the reviewer asked, why it matters
3. **Approach** - How we address it
4. **Setup** - Data loading, imports
5. **Analysis** - Interactive cells with results
6. **Summary** - Key findings, response text

## Naming Convention

- `R1_Q1_Selection_Bias.ipynb` - Referee 1, Question 1
- `R2_Temporal_Leakage.ipynb` - Referee 2, main concern
- `R3_Q8_Heterogeneity.ipynb` - Referee 3, Question 8

## Example Structure

```markdown
# R1 Q1: Selection Bias / Socioeconomic Bias

## Reviewer Question
[Exact quote from reviewer]

## Why This Matters
[Brief explanation]

## Our Approach
1. [Method 1]
2. [Method 2]
3. [Method 3]

## Results
[Interactive cells with analysis]

## Summary
[Key findings and response text]
```
